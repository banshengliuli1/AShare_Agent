# 📈 A股公告监控与资产策略 Agent

基于大模型长链推理（Long-chain Reasoning）构建的自动化金融数据监控与策略推演系统。

## 🎯 项目简介与解决痛点
在日常资产管理中，面对海量的 A 股市场公告，个人或小型团队难以第一时间提取高管减持、业绩变脸等核心信息，且难以快速测算这些变动对现有持仓复利目标的影响。

本项目通过构建 AI Agent 逻辑流，实现**“定向抓取 -> 结构化抽取 -> 结合持仓推演 -> 输出策略指令”**的闭环，将繁杂的研报阅读与投资复盘时间从每日数小时缩短至秒级，有效辅助资产配置与风险规避。

## 🧠 核心逻辑流 (Agent Pipeline)
本项目采用经典的 Agent 多步推理链路：
1. **数据抓取 (Crawler)**：定向监控用户私有股票池（Portfolio）的最新公告。
2. **节点一：信息抽取 (Extraction)**：Agent 从冗长且非结构化的公告文本中，精准提取事件类型、核心要素（如减持比例、时间窗口）及情绪倾向。
3. **节点二：深度推演 (Strategy Reasoning)**：Agent 将提取的信息与用户的**实际持仓成本价**及**既定投资策略**（如逢高减持、长期持有）相结合，进行多情景的逻辑推演，最终输出极具针对性的调仓建议。
